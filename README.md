# Portofolio App 

Aplikasi Android sederhana berbasis **Portofolio Card** yang menampilkan informasi pribadi, kontak, keahlian, dan pengalaman. Proyek ini merupakan bagian dari implementasi desain UI menggunakan XML di Android Studio.

* **Pengalaman**: Daftar proyek atau pengalaman kerja terkait pengembangan perangkat lunak.
